# Event Management System

A simple Event Management System developed using HTML, CSS, and JavaScript. This project allows users to view event details and submit event bookings through an interactive web interface.

## Features

- User-friendly interface
- Event booking form
- Input validation using JavaScript
- Responsive design
- Event information display

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Fill out the booking form and submit.

## Learning Outcomes

- Form handling with JavaScript
- Event-driven programming
- Front-end web development
- Responsive webpage design
